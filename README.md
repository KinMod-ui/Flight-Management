# Flight-Management-System


## Installation

```python
1) Download and install JAVA and MySQL and mysql-connector-java-8.0.21
2) Download all the files from the repository
3) Change username, password and url according to your MySQL
   (Source Code->src-> TableClasses -> ConnectionManager.class)
4) Copy SQL Queries from Database.doc and execute them in ur MySQL Workbench.
5) Open source code in any IDE and add mysql-connector jar file in the lib folder and run the code.
A login window will pop up.
```
